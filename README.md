# Variational Autoencoder from First Principles

A TensorFlow/Keras implementation of Variational Autoencoders (VAEs) trained on the MNIST dataset. This project investigates the probabilistic foundations of VAEs, including variational inference, ELBO optimisation, latent variable modelling, and posterior collapse mitigation through β-annealing.

## Features

- Fully Connected (FCN) VAE
- Convolutional (CNN) VAE
- β-Annealed CNN VAE
- Latent space visualisation using PCA
- KL-divergence diagnostics
- Random image generation from learned latent representations

## Overview

The model is trained by maximising the Evidence Lower Bound (ELBO):

\[
\mathcal{L}
=
\mathbb{E}_{q_\phi(z|x)}
[\log p_\theta(x|z)]
-
KL(q_\phi(z|x)\,||\,p(z))
\]

where:

- The reconstruction term encourages accurate reconstructions.
- The KL-divergence term regularises the latent space towards a standard Gaussian prior.

## Dataset

**MNIST Handwritten Digits**

- 60,000 training images
- 28×28 grayscale images
- Pixel values normalised to [0,1]
- Latent dimension: 10

## Architectures

### Fully Connected VAE

Encoder:

```text
784 → 256 → 128 → 128 → (μ, logσ²)
```

Decoder:

```text
z(10) → 128 → 128 → 256 → 784
```

### Convolutional VAE

Encoder:

```text
Input
 ↓
Conv2D(32, 3×3, stride=2)
 ↓
Conv2D(64, 3×3, stride=2)
 ↓
Dense Layers
 ↓
(μ, logσ²)
```

The decoder mirrors the latent expansion process and reconstructs the image.

### β-Annealed CNN VAE

To mitigate posterior collapse, the KL term is gradually introduced during the first 10 epochs:

```python
beta = epoch / 10
```

After epoch 10:

```python
beta = 1
```

## Results

| Model | Recon Loss | KL Loss | Total Loss |
|---------|---------:|---------:|---------:|
| Fully Connected VAE | 84.7475 | 16.8205 | 101.5300 |
| CNN VAE | 81.4763 | 18.0804 | 99.5567 |
| CNN VAE + β-Annealing | 78.8407 | 19.0787 | **97.9196** |

The CNN VAE improved reconstruction quality over the fully connected model, while β-annealing produced the best overall performance.

## Latent Space Analysis

The learned 10-dimensional latent representations were projected into 2 dimensions using PCA. The resulting latent space exhibits partial clustering of digit classes, indicating that the model learns meaningful semantic structure while maintaining a continuous latent representation.

## Generating New Digits

After training, new images can be generated by sampling from the latent prior:

```python
z ~ N(0, I)
x_hat = decoder(z)
```

The β-annealed CNN model produces the sharpest and most coherent generated samples.
